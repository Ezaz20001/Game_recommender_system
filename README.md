🎮 Game Recommender System
An intelligent Game Recommender System that suggests video games based on the user's mood and preferences. Built from scratch using machine learning models like CNN, RNN, and Random Forest, the system takes user input and generates personalized game recommendations through a simple, interactive UI.

✨ Key Features:
🧠 Mood-Based Recommendation – Suggests games based on how the user is feeling.

🕹️ Custom Dataset Support – Trained on a curated game dataset stored in Google Drive.

⚙️ Multiple ML Models – Supports CNN + RNN, Random Forest, and custom logic for better accuracy.

💬 User Input Interface – Clean UI for users to input mood and get suggestions.

📊 Explainable Output – Shows reasoning or model confidence behind each recommendation.

🛠️ Tech Stack:
Python, NumPy, Pandas

Scikit-learn, TensorFlow / PyTorch

Google Drive (for dataset storage)

HTML/CSS/JS or Streamlit (for frontend UI)
